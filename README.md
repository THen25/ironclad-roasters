# Ironclad Roasters ☕

A fictional coffee shop landing page built as a front-end portfolio project.

## Live Demo

https://then25.github.io/ironclad-roasters/

## About the Project

Ironclad Roasters is a multi-section landing page for a fictional industrial coffee shop based in Spotsylvania, VA. The goal of this project was to practice building a real-world style website using only HTML, CSS, and a small amount of vanilla JavaScript — no frameworks or libraries.

## Features

- Responsive layout that works on mobile, tablet, and desktop
- Fixed navigation bar with a hamburger menu on mobile
- Full-viewport hero section with a warm gradient overlay
- Menu section with a two-column card grid
- Photo gallery with a tall featured image slot
- Contact section with address, hours, and email
- Fade-up scroll animations using IntersectionObserver
- CSS custom properties (variables) for easy theming

## Built With

- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- Vanilla JavaScript
- Google Fonts — Bebas Neue & DM Mono

## What I Learned

- How to build a sticky nav with a working mobile hamburger menu
- Using CSS Grid for multi-column layouts that collapse on smaller screens
- Creating scroll-triggered animations with IntersectionObserver
- Layering radial gradients for background depth effects
- Organizing CSS with custom properties to keep colors consistent
